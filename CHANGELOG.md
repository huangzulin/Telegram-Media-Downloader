# Changelog

## [Unreleased]

### Added
- Initial release of Telegram Media Downloader
- Support for downloading media from Telegram channels and groups
- Web-based management interface
- WebSocket real-time updates
- SQLite database storage
- Video processing capabilities with FFmpeg
- Docker support with multi-platform builds
- GitHub Actions CI/CD pipeline

### Features
- QR code based authentication
- Download queue management
- Progress tracking and monitoring
- Thumbnail generation for videos
- RESTful API endpoints
- Health check endpoints
- Prometheus metrics support

### Technical Details
- Built with Spring Boot 3.2.5
- Java 21 runtime
- MyBatis Plus for database operations
- TDLib for Telegram client functionality
- Containerized deployment ready
